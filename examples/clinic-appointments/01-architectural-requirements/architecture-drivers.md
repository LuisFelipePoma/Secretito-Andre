## Intake Summary

The clinic chain needs to replace manual appointment management with a web scheduling system. The business wants fewer double-bookings, faster appointment handling, fewer missed appointments and better schedule visibility.

## System Requirements

| ID | Requirement | Description |
| --- | --- | --- |
| REQ-001 | Register and log in | Patients and staff access the system with email/password for MVP. |
| REQ-002 | Search availability | Patients/receptionists search availability by specialty, location, doctor and date. |
| REQ-003 | Book appointment | A user books an available slot. |
| REQ-004 | Reschedule/cancel appointment | A user changes or cancels an existing appointment. |
| REQ-005 | Manage calendars | Receptionists manage doctor calendars and blocks. |
| REQ-006 | View daily schedule | Doctors see their schedule for the day. |
| REQ-007 | Send notification request | System asks an external provider to send email/SMS notifications. |
